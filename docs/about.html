О проекте MetricForge

Назначение

MetricForge — это учебная практика по лабораторным работам по дисциплине «Метрология, стандартизация и сертификация в информационных технологиях». Цель — изучить методы оценки качества и сложности программного кода.

Практика

Первая лабораторная работа заключается в разработке программы-парсера с графическим интерфейсом для анализа исходного кода на языке F#. Программа рассчитывает метрики Холстеда (6 базовых и 3 расширенные) и выводит результат в виде таблицы.

Технологии

  • Python
  • FastAPI (Backend API)
  • Jekyll (статическая генерация сайта)
  • HTML5 + Bootstrap 5
  • JavaScript (Fetch API)
  • jQuery
  • GitHub Pages (Frontend-хостинг)
