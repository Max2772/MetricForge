О проекте MetricForge

Назначение

MetricForge — веб-приложение для анализа программного кода на F# с расчётом метрик Холстеда. Система позволяет оценивать сложность, объём и трудоёмкость программ.

Как это работает

Пользователь вставляет исходный код в редактор. После отправки запроса сервер анализирует операторы и операнды, вычисляет метрики Холстеда и возвращает результат в виде таблицы.

Технологии

  • Python
  • FastAPI (Backend API)
  • Jekyll (статическая генерация сайта)
  • HTML5 + Bootstrap 5
  • JavaScript (Fetch API)
  • jQuery
